# Design

Візуальна система артефактів Кутка (research-сторінки; основа для майбутнього UI продукту).
Згенеровано як seed при редизайні `research/index.html`; перегенерувати з реального коду: `/impeccable document`.

## Theme

Світла, «паперова». Сцена: дизайнер чи замовник читає ресерч удень на ноутбуці або з телефона в метро — робочий документ, довге читання. Темна тема тут не виправдана.

## Color (OKLCH, стратегія Restrained)

| Token | Value | Роль |
|---|---|---|
| `--paper` | `oklch(97.5% 0.006 75)` | фон, теплий папір |
| `--ink` | `oklch(25% 0.012 60)` | основний текст |
| `--muted` | `oklch(48% 0.015 65)` | вторинний текст |
| `--line` | `oklch(90% 0.008 75)` | hairline-розділювачі |
| `--line-soft` | `oklch(93.5% 0.006 75)` | внутрішні лінії таблиць |
| `--clay` | `oklch(52% 0.115 40)` | акцент: лінки, активна навігація, маркери (≤10% поверхні) |
| `--clay-soft` | `oklch(94.5% 0.02 50)` | тінт для виділених блоків |

Жодних чистих `#000`/`#fff`; всі нейтрали підігріті до теплого hue (~60–75). Один акцент — теракота («куток», цегла, дім).

## Typography

- **Manrope** (Google Fonts, кирилиця) — все: заголовки 700–800, текст 400–500.
- **JetBrains Mono** — мітки, нумерація, цифри, статуси: 11px, uppercase, letter-spacing .08em.
- Шкала (product, ratio ~1.2): 11 / 13.5 / 15.5 (body) / 19 / 22 / 30.
- Міра прозового рядка ≤ 72ch.

## Layout

- Одна колонка, max-width 840px; таблиці можуть виходити в горизонтальний скрол.
- Без карток за замовчуванням: секції розділяє простір + hairline. Тінтовані блоки — лише для «вибір» / «не спрацює».
- Sticky top-nav з якорями і scroll-spy; на мобільному — горизонтальний скрол навігації.
- Нумерація секцій і пунктів — mono, clay.

## Motion

150–250ms, тільки стан (hover/focus/active + scroll-spy). `prefers-reduced-motion` вимикає smooth-scroll. Жодних page-load хореографій.

## Accessibility

WCAG AA: контрасти перевірені (muted ≈ 6:1, clay ≈ 5.5:1 на paper), skip-link, `:focus-visible`, семантичні landmarks, `scope` у таблицях, alt у скрінів, таргети ≥44px.
